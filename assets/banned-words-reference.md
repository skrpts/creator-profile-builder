# Default Banned Language Reference

This is the starting list. During profile building, each entry is checked against the creator's actual content. Words the creator genuinely uses are removed. Words they avoid that are not on this list are added. The result is a personalised banned list in the Voice Profile.

## AI Cliché Words

Words that signal generic AI-generated content. Most human writers rarely use these.

| Word | Why it's banned | Common replacement |
|------|----------------|-------------------|
| delve | AI favourite, rarely used in natural writing | explore, examine, look at |
| dive (into) | Overused metaphor, especially "deep dive" | examine, investigate, cover |
| unlock | Implies magic, vague | enable, allow, make possible |
| unleash | Hyperbolic | release, enable, use |
| harness | Overused in tech/business writing | use, apply, take advantage of |
| leverage | As a verb — corporate jargon | use, build on, take advantage of |
| streamline | Vague improvement claim | simplify, speed up, reduce steps |
| optimise | Often used without specifying what improves | improve, speed up, reduce [specific metric] |
| empower | Vague, patronising | enable, give [specific capability] |
| elevate | Vague uplift metaphor | improve, strengthen, raise [specific quality] |
| robust | Meaningless without context | reliable, thorough, tested, resilient |
| seamless | Nothing is seamless | smooth, integrated, straightforward |
| transformative | Hyperbolic | significant, useful, effective |
| pivotal | Overused emphasis | important, critical, key |
| nuanced | AI uses this to sound sophisticated | detailed, specific, complex |
| holistic | Vague buzzword | complete, end-to-end, comprehensive |
| synergy | Corporate jargon | combined effect, collaboration, overlap |
| paradigm | Pretentious | approach, model, framework |
| landscape | Metaphorical overuse — "the AI landscape" | field, space, market, industry |
| tapestry | AI metaphor favourite | mix, combination, collection |
| intricate | AI favourite for "complex" | complex, detailed, layered |
| utilise | Formal for no reason | use |
| facilitate | Corporate speak | help, enable, run, organise |
| endeavour | Unnecessarily formal | try, attempt, work, effort |
| myriad | AI favourite | many, numerous, a range of |
| plethora | Same problem as myriad | many, plenty, a wide range of |
| realm | "In the realm of" — AI padding | field, area, domain |
| foster | Corporate/academic speak | encourage, build, support |
| bolster | AI favourite | strengthen, support, reinforce |
| cornerstone | Overused metaphor | foundation, basis, key part |

## AI Cliché Phrases

Phrases that make content sound templated or machine-generated.

| Phrase | Why it's banned | What to do instead |
|--------|----------------|-------------------|
| In today's [fast-paced/digital/modern] world | Generic opener, pure padding | Start with the actual point |
| It's worth noting that | Filler — if it's worth noting, just note it | Delete and state the thing directly |
| At the end of the day | Cliché summary phrase | Use a specific conclusion |
| When it comes to | Filler transition | Delete, or use "For" / "With" |
| In order to | Verbose — "to" does the same job | Replace with "to" |
| The fact that | Usually deletable filler | Rephrase without it |
| It goes without saying | Then don't say it | Delete entirely |
| Needless to say | Same problem | Delete entirely |
| As we all know | Presumptuous | Delete, or "Most teams already..." |
| In conclusion | Mechanical essay signpost | Use a natural closing sentence |
| Let's dive in | AI/YouTube opener cliché | Start with the content |
| Without further ado | Padding | Delete |
| Game-changer | Hyperbolic, vague | Explain the specific impact |
| Paradigm shift | Pretentious | Describe the actual change |
| Best practices | Often means "things I'm about to list" | Name the specific practices |
| Take it to the next level | Vague aspiration | State the specific improvement |
| Think outside the box | Ironic cliché | Describe the actual creative approach |
| Move the needle | Corporate metric-speak | State what actually changes |
| Low-hanging fruit | Overused prioritisation metaphor | "Quick wins", or name the specific items |
| Circle back | Corporate speak | "Revisit", "return to" |
| Touch base | Same | "Check in", "follow up" |
| Seamless integration | Nothing integrates seamlessly | "Works with", "connects to" |
| Enterprise-grade | Meaningless without definition | State the specific capability |
| Industry-leading | Self-congratulatory, unverifiable | State the specific advantage |
| Cutting-edge | Vague modernity claim | State what's actually new |
| Breakthrough | Hyperbolic | State the specific advance |
| Revolutionary | Almost never true | State the specific change |

## Empty Intensifiers

Words that add emphasis without adding meaning.

| Word | When to ban | When it's OK |
|------|------------|-------------|
| very | Almost always deletable | Direct quotes, dialogue |
| really | Same | Emphasis in casual voice ("this really matters") |
| extremely | Usually hyperbolic | Measured technical claims ("extremely low latency = <1ms") |
| incredibly | AI favourite intensifier | Rare — only when genuinely incredible |
| absolutely | Filler emphasis | Contradicting a claim ("Absolutely not") |
| literally | Figurative usage is wrong; literal usage is redundant | Never, unless distinguishing from figurative |
| essentially | Usually filler | Simplifying a technical concept for a lay audience |
| fundamentally | Usually filler | Genuine paradigm-level distinction |
| basically | Condescending in writing | Casual speech patterns |

## Hedging Crutches

Over-qualification that weakens writing.

| Phrase | Problem | Fix |
|--------|---------|-----|
| I think | Redundant in opinion pieces — the whole piece is what you think | Delete, or use "My experience suggests" for nuance |
| In my opinion | Same | Delete |
| It could be argued that | Passive hedge | Argue it directly, or attribute the argument |
| Some might say | Weasel phrase | Name who says it, or state it directly |
| Perhaps | Weakens the claim — either commit or cut | Use sparingly, only for genuine uncertainty |
| Arguably | Hedge that adds nothing | Delete and make the argument |

## Calibration Notes

During profile building, each entry above is checked against the creator's content:

1. **Remove from ban list** if the creator uses the word/phrase naturally and it's part of their genuine voice (e.g., a finance writer may legitimately use "leverage")
2. **Add to caution list** if the creator uses it rarely — flag for review but don't auto-reject
3. **Add new entries** if the creator consistently avoids words/phrases not on this list
4. **Add replacements** based on what the creator actually says instead of each banned term
