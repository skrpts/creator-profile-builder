## Usage Guide

How to use this profile when writing as or for this creator:

- **For AI-assisted drafting:** Feed the Voice Summary and Voice Markers sections as style context. Add the Banned Language section as a constraint.
- **For human ghostwriters:** Read the full profile. Focus on Sentence Architecture and Structural Habits for the scaffolding, then Vocabulary Fingerprint for word-level accuracy.
- **For editorial review:** Check output against Anti-Patterns and Banned Language. Verify Voice Markers are present.
- **For tone adaptation:** Combine this Voice Profile with the appropriate Audience Profile via step.context to calibrate formality and depth.
